Current Narrative / Operating Posture

The main roadmap is zone-control RL MVP: test whether an RL agent can reduce mega-fires in a toy forest-fire model better than honest baselines without clearing the forest or over-spending interventions. The prior single-cell switch-point efficacy work is parked as diagnostic/prototype infrastructure.

Next gate: issue #54 action contract fixtures — deterministic zone indexing, no-op, one-zone thinning, and decision interval semantics. Do not jump to training or rendering.

Do Not Do

  • Do not run GPU/Puffer/training/render/raylib/c_render work on the VPS.
  • Do not run local WSL HITL commands without Jamie approving the exact command.
  • Do not claim final efficacy, SOC-control, publication-grade science, policy value, or real-wildfire validity.
  • Do not delete parked switch-point artifacts.
  • Do not implement from parent tracker alone; follow child issue scope.
  • Do not overwrite README.md or docs/PRD.md without explicit instruction.

Fallback Rules

  1. If stale (>14 days), re-verify issue #53, issue #54, current PRD, and repo state.
  2. If question touches old efficacy path, route to parked diagnostics and label it parked.
  3. If question asks for real-world wildfire claims, refuse as out of scope.
  4. If no match, fall back to Project Board then Projects MOC.
