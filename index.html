Pixiedust Knowledge Wiki

A domain-first human surface for Pixiedust's agent-readable knowledge packs and concepts. Humans browse domains; agents use domain, collection, type, and role metadata.

Covers

Active KPR packs, Knowledge concepts, source/routing boundaries, project/knowledge/MOC entrypoints, and current freshness posture.

Not covered

Raw private vault content, secrets, memory dumps, runtime automation internals, MCP/search/RAG/confidence calibration, or automatic pack rewriting.

Current as of

Generated from checked-in KPR packs and Knowledge concepts. Markdown packs remain source/routing truth; Knowledge concept notes remain source truth.

🤖 Agent access: start with root llms.txt, then follow domain and local node contracts.
